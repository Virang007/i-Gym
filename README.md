# Full Stack I-GYM Application

A comprehensive Full Stack Gym Application built with React, Node.js, MongoDB, and Express. This application features user authentication, workout tracking, and email functionality.

## Features

- **Full Stack Architecture:** Utilizes React for the front end, Node.js and Express for the back end, and MongoDB for the database.
- **User Authentication:** Secure login and registration system.
- **Workout Tracking:** Users can track their workouts and progress.
- **Email Functionality:** Email notifications for user actions and updates.

## Technologies Used

- **Frontend:** React
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Email:** NodeMailer or similar email service
